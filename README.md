# Logistics & Operations Dashboard

This Power BI dashboard provides insights into logistics and operations performance through interactive visualizations and KPI tracking.

## Tools Used

- SQL
- Microsoft Excel

## Key Features
- Order & shipment analysis
- Operational KPI tracking
- Interactive filters and slicers

- Business performance insights
